## 使用前准备

### 1. 数据库迁移（必需）
```bash
cd /www/wwwroot/fakabot
alembic upgrade head
```

### 2. 配置环境变量（可选）

**Telegram 通知（可选）:**
```bash
# .env
MASTER_BOT_TOKEN=你的Bot Token
```

**支付配置（租户级，通过 Admin Web 配置）:**
- EPUSDT: 每个租户配置自己的商户ID和密钥
- 易支付: 每个租户配置自己的PID和密钥

**外部货源（租户级，通过 Admin Web 配置）:**
- mcy_shop: 每个租户配置自己的凭据
- standard_http: 每个租户配置自己的凭据

### 3. 启动服务
```bash
# 启动 Worker
python -m workers.report_worker
python -m workers.subscription_worker
python -m workers.payment_retry_worker

# 或使用启动脚本
./scripts/start_service.sh
```

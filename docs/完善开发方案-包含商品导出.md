#### 生命周期状态机

```
trial → active (付款) / grace (未付款)
      ↓
active → active (续费) / grace (未续费)
       ↓
grace → active (补付款) / suspended (宽限到期)
      ↓
suspended → (数据保留期)
          ↓
    (标记待清理，不实际删除)
```

#### 处理逻辑

**1. 试用到期检查（每小时）**
```python
async def process_trial_ending():
    # 查询 trial_ends_at <= now + 1天 的订阅
    # 发送 Telegram 提醒（试用即将到期）
    
async def process_trial_ended():
    # 查询 trial_ends_at <= now 且 status=trial
    # 检查是否有付款
    # 有付款 → active，无付款 → grace
    # 发送 Telegram 通知
    # 写审计日志
```

**2. 当前周期结束检查（每小时）**
```python
async def process_period_ending():
    # 查询 current_period_ends_at <= now + 3天
    # 发送 Telegram 续费提醒
    
async def process_period_ended():
    # 查询 current_period_ends_at <= now 且 status=active
    # 检查是否有续费订单
    # 有续费 → 延长周期，无续费 → grace
    # 发送通知和审计
```

**3. 宽限期到期检查（每小时）**
```python
async def process_grace_ended():
    # 查询 grace_ends_at <= now 且 status=grace
    # 检查是否补付款
    # 未付款 → suspended
    # 清理 Webhook 缓存
    # 发送暂停通知
```

**4. 保留期到期检查（每天）**
```python
async def process_retention_ended():
    # 查询 data_retention_until <= now
    # 标记为待清理（不实际删除数据）
    # 发送最后通知
```

#### 重试策略

**退避时间表：**
```
第1次重试: 1分钟后
第2次重试: 5分钟后
第3次重试: 30分钟后
超过3次: 不再重试
超过24小时: 不再重试
```

**实现逻辑：**
```python
async def retry_failed_callbacks():
    # 查询条件：
    # - status = 'failed'
    # - created_at > now - 24h
    # - retry_count < 3
    # - last_retry_at IS NULL OR last_retry_at < now - 退避时间
    
    for callback in failed_callbacks:
        try:
            # 调用 OrderPaymentService.process_payment_callback()
            # 成功 → 更新 status = 'success'
            # 失败 → retry_count += 1, last_retry_at = now
        except Exception as e:
            # 记录错误但不中断循环
            logger.error(f"Retry callback {callback.id} failed: {e}")
```

### 1.4 Worker 进程管理 (0.5天)

#### Worker 启动脚本

```python
# workers/base.py
class BaseWorker:
    def __init__(self, name: str, interval: int):
        self.name = name
        self.interval = interval
        self.running = False
        
    async def start(self):
        self.running = True
        while self.running:
            try:
                await self.process()
            except Exception as e:
                logger.error(f"{self.name} error: {e}")
            await asyncio.sleep(self.interval)
    
    async def stop(self):
        self.running = False
    
    @abstractmethod
    async def process(self):
        pass

# workers/report_worker.py
python -m workers.report_worker

# workers/subscription_worker.py  
python -m workers.subscription_worker

# workers/payment_retry_worker.py
```

```python
@router.get("/health")
async def health_check():
    return {
        "status": "healthy",
        "timestamp": datetime.now().isoformat(),
        "version": "1.0.0",
    }

@router.get("/health/detailed")
async def detailed_health_check():
    checks = {
        "database": await check_database(),
        "redis": await check_redis(),
        "workers": await check_workers(),
        "storage": await check_storage(),
    }
    all_healthy = all(checks.values())
    status_code = 200 if all_healthy else 503
    return JSONResponse(
        status_code=status_code,
        content={
```
